# Agent Instructions

## Build & Release

- **Do NOT modify files in the `release/` folder directly.**
- The `release/` folder is generated and updated only via the `make release` command.
- When making changes, edit the source files (`index.html`, `styles.css`, `main.js`, etc.) in the project root instead.
- After changes are complete, `make release` will regenerate the `release/` folder from the updated source files.
